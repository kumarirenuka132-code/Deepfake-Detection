# 🛡️ Deepfake Detection using Vision Transformer (ViT)

## 📌 Project Overview

This project detects whether an uploaded image or video is **Real** or **Fake** using a Vision Transformer (ViT) model. The application is developed using **PyTorch** and **Streamlit** and is trained on the **Celeb-DF v2** dataset.

---

## 🚀 Features

- Image Deepfake Detection
- Video Deepfake Detection
- Face Detection using OpenCV
- Face Cropping and Preprocessing
- Vision Transformer (ViT) Model
- Confidence Score
- User-Friendly Streamlit Interface

---

## 🛠️ Technologies Used

- Python
- PyTorch
- Vision Transformer (ViT)
- OpenCV
- Streamlit
- NumPy
- Pillow

---

## 📂 Dataset

**Dataset:** Celeb-DF v2

## 🤖 Trained Model

The trained model is not included in this repository because it is larger than GitHub's upload limit.

Download the trained model from Google Drive:

**🔗 Model Download:**

https://drive.google.com/file/d/1O0uOM6P_4nRitUm67FAWwRLV6U24C4DV/view?usp=drive_link

After downloading, place the model file in the project folder:

```
Deepfake-Detection/
│── app.py
│── deepfake_vit_model.pth
│── requirements.txt
│── README.md
```

## 📦 Installation

Install the required libraries:

```bash

```
